Portfolio

Weather Website Template - November 2023

This website was originally used to research how to implement features into the final assessment piece we were tasked to make for the T Level course. Since finishing the course, I have fleshed it out further into a full weather forecast website, air quality forecast, placeholder articles and other features. There is even a login system for the website that hashes the user's password, though the email isn't checked to confirm it is real, so you don't have to put personal information on this website, as the login is only used for the health tracker feature.

This website was made primarily using PHP, with weather API's such and openweathermap, Froala text editor and tomorrow.io.

The Drop - May 2023

This is a unity game concept made within a week as part of a game jam that we did at DeMonfort University on work experience. This was developed by me and a friend on the work experience course.

This is a simple dropper game that involves the player falling through a map, avoiding randomly generated obstacles, with a dash ability for avoiding harder obstacles and ring powerups that will speed up or slow down the drop rate of the player.

Both me and my friend worked on the code for this game. I also worked on most of the models, excluding the cloud models, which are from The Fallout Nerd's 3LE Low Poly Cloud Pack. While this game is very barebones, there were many features of this game that was a challenge to develop, such as the level generation.
